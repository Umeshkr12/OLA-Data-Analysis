 


# OLA Data Analysis Project

## Description
This project involves analyzing OLA ride data using Power BI, SQL, and Excel. The goal is to uncover key insights, trends,and patterns to help make data-driven decisions. 
The analysis includes data extraction and querying in SQL, preliminary processing in Excel, and visualization in Power BI.

## Technologies Used
- **Power BI**: For creating interactive dashboards.
- **SQL**: For querying and analyzing data.
- **Excel**: For data preprocessing and cleaning.

## Features
- Insights into top-performing cities and frequent users.
- Analysis of average Customer Ratings by Vehicle Type.
- Interactive dashboards with filters for detailed exploration.
- Revenue by Payment Method.
- cancelled Rides Reasons.
- Top 5 Vehicle Types by Ride Distance.

## How to Use
1. Download the repository to your local machine.
2. Open the provided SQL scripts (`.sql` files) in your SQL database tool to explore the queries and analysis.
3. Open the Power BI dashboard (`.pbix` file) using Power BI Desktop to view the visualizations.
4. Follow any additional instructions mentioned in the comments or README for specific steps.
